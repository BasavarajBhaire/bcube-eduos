# Quality Assurance Engine Rule Index

Owner: QA Lead  
Prefix: `QA-`

## Active foundation rules

- `QA-GATE-001` — Every production artifact MUST pass mandatory quality gates.
- `QA-EDU-001` — Educational compliance MUST be validated.
- `QA-PUB-001` — Publishing and print compliance MUST be validated.
- `QA-SCORE-001` — Gold release requires a minimum score of 90/100 and zero critical defects.
- `QA-AUDIT-001` — Validation evidence MUST be retained for audit.

Outputs: validation report, defect log, Gold score, release decision, and audit record.
