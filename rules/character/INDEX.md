# Character Engine Rule Index

Owner: Character & Brand Lead  
Prefix: `CE-`

## Active foundation rules

- `CE-STAR-001` — Star mascot identity, colours, proportions, and behaviour MUST remain canonical.
- `CE-POSE-001` — Poses MUST communicate purposeful learning actions.
- `CE-EXP-001` — Expressions MUST match the intended learning emotion.
- `CE-DIV-001` — Character representation MUST remain inclusive and respectful.
- `CE-VAL-001` — Character validation MUST pass before QA release.

Outputs: character ID, role, pose, expression, wardrobe, interaction, and continuity tags.
