# Teaching Engine Rule Index

Owner: Teacher Academy Lead  
Prefix: `TE-`

## Active foundation rules

- `TE-FAC-001` — Every activity MUST define a teacher facilitation strategy.
- `TE-QUE-001` — Teacher prompts SHOULD use open-ended questioning.
- `TE-SCAF-001` — Scaffolding SHOULD progress from modelling to guided practice to independence.
- `TE-OBS-001` — Observation focus MUST identify evidence of learning.
- `TE-VAL-001` — Teaching validation MUST pass before QA approval.

Outputs: facilitation strategy, teacher action, question prompt, observation focus, and feedback cue.
