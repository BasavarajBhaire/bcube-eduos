# Illustration Engine Rule Index

Owner: Illustration Director  
Prefix: `IE-`

## Active foundation rules

- `IE-STYLE-001` — Artwork MUST use the approved BCube preschool illustration style.
- `IE-SCENE-001` — Each page MUST show one coherent educational scene.
- `IE-LIGHT-001` — Lighting MUST remain warm, clear, and print-friendly.
- `IE-PROP-001` — Props MUST support the learning objective.
- `IE-VAL-001` — Illustration validation MUST pass before Character and QA review.

Outputs: scene type, environment, rendering style, lighting, perspective, props, and negative constraints.
