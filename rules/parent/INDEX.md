# Parent Partnership Engine Rule Index

## Active foundation rules

- `PP-COM-001` — Parent guidance MUST use clear, encouraging language.
- `PP-RES-001` — Home activities SHOULD use common low-cost materials.
- `PP-TIME-001` — Activity duration MUST match the developmental level.
- `PP-VAL-001` — Parent Partnership validation MUST pass before release.
