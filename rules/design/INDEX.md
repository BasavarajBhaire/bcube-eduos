# Design Engine Rule Index

Owner: Design Lead  
Prefix: `DE-`

## Active foundation rules

- `DE-GRID-001` — Every page MUST use an approved BCube layout grid.
- `DE-TYPE-001` — Typography MUST follow the approved hierarchy and minimum readability rules.
- `DE-SPACE-001` — Spacing MUST use approved design tokens.
- `DE-TPL-001` — Every page MUST inherit an approved template ID.
- `DE-VAL-001` — Design validation MUST pass before Visual Grammar execution.

Rules inherit Publishing Engine geometry and emit normalized layout metadata for downstream engines.
