# Visual Grammar Engine Rule Index

Owner: Visual Design Lead  
Prefix: `VG-`

## Active foundation rules

- `VG-FOC-001` — Every page MUST contain one dominant educational focal point.
- `VG-FLOW-001` — Eye flow SHOULD lead from title to learning scene to activity.
- `VG-COG-001` — Decorative content MUST NOT increase unnecessary cognitive load.
- `VG-WHITE-001` — White space SHALL be treated as an instructional resource.
- `VG-VAL-001` — Visual Grammar validation MUST pass before illustration compilation.

Outputs: focal hierarchy, eye-flow map, visual-balance metadata, and clutter constraints.
