# BCube Gold Production v4

This directory contains the locked BCube Gold page-design and artwork-production standard for Nursery, LKG and UKG. Existing `production-prompts` v2/v3 curriculum packages remain unchanged and are the curriculum source of truth.

## Canonical source hierarchy

1. `production-prompts/README.md` — official 30-book portfolio plan.
2. Each book release README — official 41-page sequence, prompt IDs and page titles.
3. Each page Markdown/JSON package — official page objective, visible wording, activity, teacher move, parent extension, illustration requirements and prohibitions.
4. `BCube_Gold_Production_v4/` — visual-production layer only; it must not rewrite curriculum, chapters, page order or learning objectives.

## Locked portfolio

- Nursery: 10 books × 41 pages
- LKG: 10 books × 41 pages
- UKG: 10 books × 41 pages
- Total: 30 books / 1,230 page packages

See: [`../production-prompts/README.md`](../production-prompts/README.md)

## Locked production principles

- Preserve the existing book plan, chapters, page titles, page numbering and learning objectives.
- Improve only page design, illustration quality, teaching clarity and print production.
- Use the official BCube logo asset only; never ask an image model to redraw it.
- Reserve a fixed clean logo zone during artwork generation, then place the official logo in post-production.
- One clear learning focus per page.
- Maximum two child-facing activities unless the source page explicitly requires otherwise.
- Large focal illustration, generous white space and clear teacher–child interaction.
- A4 portrait, print-first, safe margins and publication-ready hierarchy.
- Design rule: Less. Bigger. Better.

## Alphabet-page clarification

The canonical 30-book plan currently contains no standalone book named `Alphabet Champions`. Alphabet-related learning is present inside the approved literacy releases, including `production-prompts/early-literacy-adventures/lkg/v3/` (for example Alphabet Review, Uppercase & Lowercase, Letter Sounds and Beginning Sounds). Any A–Z single-letter page series produced under v4 is therefore treated as a **supplemental artwork set** until it is explicitly added to the curriculum plan. It must not replace or silently modify an existing 41-page book.
