# BCube Gold Production v4

This directory contains the locked BCube Gold publishing standard for Nursery, LKG and UKG. Existing v2 and v3 remain unchanged.

## Principles
- Preserve curriculum, chapters and learning objectives.
- Improve only page design and production quality.
- Official logo only (never AI-generated).
- One learning objective per page.
- Less. Bigger. Better.
- Print-first A4 design.
