# BCube Gold Acceptance Criteria v1.0

Status: FOUNDER LOCKED
Version: Gold Production v4.1

## No-Compromise Rule

No page is approved because it is "good enough". A page is approved only when every critical rule passes and the weighted score is at least 95/100. Any failed critical rule forces rejection and regeneration.

## Critical identity rules

- Book, level, page number, prompt ID, title and chapter/module must exactly match the canonical repository.
- Visible wording, activity, illustration scene and response space must come only from the page Markdown and JSON package.
- No invented book, page, title, label, activity, teacher box, parent box, speech balloon or instructional text.
- The official BCube logo asset must be used unchanged. Never redraw, recolour, crop, stretch or approximate it.
- Cover pages must not show a visible page number or worksheet elements unless the canonical package explicitly requires them.
- A final book must use the complete 43-package front-matter architecture; mixed 41/43 structures are critical failures.
- About, Copyright and both Contents pages must hide their internally counted numbers 1–4.
- Welcome must be the first visibly numbered page and must show 5.
- Contents must occupy two pages, use canonical module headings and exclude front matter.

## Single-page output rule

Every generation must contain exactly one flat, front-facing A4 portrait page.

Automatically reject:

- collages,
- contact sheets,
- preview boards,
- storyboards,
- book summaries,
- multi-page composites,
- mockups,
- perspective pages,
- page curls,
- device frames,
- hands holding the page.

## Geometry and print rules

- A4 portrait, 210 × 297 mm.
- 3 mm bleed.
- Minimum 10 mm safe margin.
- 12 mm binding margin.
- 300 DPI target.
- CMYK-safe palette.
- No clipped text, cropped key objects or content outside safe zones.

## Layout and readability

- One dominant focal scene.
- Generous breathing space; no overcrowding.
- No tiny text, dense paragraphs or decorative clutter.
- Illustration normally occupies 55–70% of the page unless the canonical page type requires otherwise.
- Activities must be large, obvious and developmentally appropriate.
- Tracing paths must support preschool pencil control.
- Colouring areas must have large enclosed regions and clean outlines.

## Character and teaching quality

- Star must retain the locked bright-yellow rounded five-point identity, expressive face, blue shoes and small blue cape.
- Teacher-child interaction must be warm, natural, purposeful and connected to the page objective.
- Children must show age-appropriate proportions, natural expressions, inclusive Indian diversity and correct anatomy.
- One page must communicate one clear learning focus unless the canonical package explicitly requires more.

## Automatic critical rejection

Reject immediately for any of the following:

- wrong book, level, page, prompt ID or title,
- more than one page in the image,
- AI-generated or altered BCube logo,
- wrong age badge,
- invented or missing visible text,
- invented or missing activity,
- wrong page-number policy,
- one-page Contents or Contents that lists Cover, About, Copyright or Contents,
- any visible number on P002–P005,
- Welcome numbered other than 5,
- cover containing worksheet elements,
- inconsistent Star identity,
- tiny unreadable text,
- overcrowded layout,
- unsafe or frightening imagery,
- anatomy defects,
- cropped key content,
- any unresolved source conflict.

## Approval levels

- 100: Founder Masterpiece
- 95–99: BCube Gold Certified
- 90–94: Reject for refinement
- Below 90: Automatic reject

A critical failure always overrides the numeric score.
