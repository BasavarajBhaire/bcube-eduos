# Approved Communication Champions Pages

Store only approved, individual 2480 × 3508 PNG pages here.

Each page uses a hash-addressed filename:

```text
CC-NURSERY-V4-P023-<sha256-prefix>.png
```

The adjacent `asset-lock.json` records the full SHA-256, source-package version, reviewer, approval time and QA-record path. Do not copy candidate or rejected images into this directory manually.
