## Learning objective
Recognize, say, trace, and write own name.

## Exact visible wording
My name is ________. Trace my name. Write my name.

## Child action
Say: My name is ____. Trace the adult-written name once, then copy it once on the second line.

## Observable evidence
Child says own name and makes an adult-supported tracing or copying attempt.

## Teacher move
Write the child’s name in the trace line, model the sentence once and invite the child to say it.

### Teacher question
- What is your name?

## Parent connection
Say family members’ names and clap the syllables in each name.

## Locked composition
One child faces a mirror while Star holds a name card; two wide horizontal name lines occupy the protected response area.

## Exact response space
One dotted trace line and one blank copy line.
