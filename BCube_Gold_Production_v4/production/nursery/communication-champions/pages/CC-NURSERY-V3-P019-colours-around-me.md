## Learning objective
Recognize and name basic colors.

## Exact visible wording
Find: red, blue, yellow, green. Circle the objects. Colour the swatches.

## Child action
Find and circle one red, blue, yellow and green object in the scene, then colour four blank swatches to match.

## Observable evidence
Child identifies four basic colours and completes matching swatches.

## Teacher move
Model one colour search using the red ball, then let the child find the remaining colours.

### Teacher question
- What colour did you find?

## Parent connection
Find one object of each target colour during a room walk.

## Locked composition
Simple playroom with exactly four focal objects: red ball, blue bag, yellow cup, green plant; four blank swatches below.

## Exact response space
Four circle targets and four blank colour swatches.
