# BCube Gold Publishing Standard v5.0

## Canonical rules

- Nursery contains exactly ten official books in the order defined by `books/catalog.json`.
- Each official Nursery release contains 41 pages unless the catalog version is formally changed.
- Prompt IDs use `<CODE>-NURSERY-V5-P###`.
- Titles, slugs, IDs, manifests, workbooks, workflows and release notes must agree with the catalog.
- Development-only titles are archived and cannot appear in the official Nursery release index.
- Artwork, human visual approval, prepress proof and publisher sign-off remain mandatory release gates.
