# BCube Gold Publishing Standard v5.1

## Canonical rules

- Nursery contains exactly ten official books in the order defined by `books/catalog.json`.
- Each official Nursery release contains exactly 44 pages.
- Page architecture: 1 front cover + 1 inside cover/book-information page + 1 ownership page + 40 learning/support pages + 1 back cover.
- Certificate, parent partnership, review and celebration content must be planned within the 40 learning/support pages where applicable.
- Prompt IDs use `<CODE>-NURSERY-V5-P001` through `<CODE>-NURSERY-V5-P044`.
- Titles, slugs, IDs, manifests, workbooks, workflows and release notes must agree with the catalog.
- Development-only titles are archived and cannot appear in the official Nursery release index.
- Artwork, human visual approval, prepress proof and publisher sign-off remain mandatory release gates.
