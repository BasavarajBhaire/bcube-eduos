# Supplemental Alphabet Pages — Production Plan

Status: Visual-production supplement; not part of the canonical 30-book / 1,230-page curriculum plan.

## Purpose

Create a consistent A–Z preschool page set using the approved A-page visual benchmark. This set remains separate from the existing 41-page books until curriculum governance formally assigns it to a book and page sequence.

## Locked page architecture

1. Fixed blank logo zone at top-left for later placement of the exact official BCube logo.
2. Large centred page title: `<LETTER> is for <OBJECT>`.
3. One short adult-readable learning objective.
4. One dominant hero scene occupying roughly 50–55% of the page.
5. Bottom-left activity: trace uppercase and lowercase letter.
6. Bottom-right activity: colour the hero object.
7. Footer prompt led by Star: `<LETTER> says /sound/. Can you say it with me?`
8. A4 portrait, white background, safe margins, rounded preschool typography, thick clean outlines and generous breathing space.

## Character and branding rules

- Do not generate or redraw the BCube logo inside the artwork.
- Keep a clean reserved logo area; composite the official logo afterward.
- Star is the recurring guide, supporting the child rather than competing with the main learning object.
- Use warm teacher–child or child–Star interaction where it improves teaching clarity.
- Keep one dominant focal object and no decorative clutter.

## QA gate for every page

- Correct uppercase/lowercase letter.
- Correct hero word and phonics prompt.
- No AI-generated BCube logo.
- Trace area large enough for preschool pencil control.
- Colouring object has large enclosed regions and clean outlines.
- No tiny text, crowding, duplicate focal objects or ambiguous instructions.
- Important content remains inside A4 safe margins.
