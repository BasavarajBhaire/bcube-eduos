# Module Structure

## Module 1 — My Creative World (Pages 5–11)
Focus: free marks, imaginative completion, visual storytelling and confidence.

## Module 2 — Shapes and Patterns (Pages 12–18)
Focus: shape recognition, transformation, repetition, sequencing and design.

## Module 3 — Colours Everywhere (Pages 19–25)
Focus: colour matching, expressive choice, contrast, mood and visual preference.

## Module 4 — Build and Create (Pages 26–32)
Focus: parts-to-whole thinking, spatial arrangement, construction and redesign.

## Module 5 — Creative Nature (Pages 33–39)
Focus: observation of natural forms, symmetry, texture, pattern and reinterpretation.

## Module 6 — My Big Ideas (Pages 40–43)
Focus: invention, open-ended problem solving, story creation and presentation.

## Module rhythm
Each module follows a common rhythm:
1. Notice
2. Try
3. Change
4. Combine
5. Make
6. Share
7. Reflect

This recurring rhythm supports predictable classroom facilitation while keeping the creative outcomes open-ended.
