# Problem-Solving Progression Matrix

| Stage | Child behaviour | Adult support |
|---|---|---|
| Observe | Notices visible clues | Ask what the child sees |
| Think | Compares possibilities | Offer wait time |
| Try | Tests an idea | Welcome experimentation |
| Solve | Chooses and acts | Ask why it may work |
| Express | Shares the process | Praise effort and explanation |
