# Design System

- Format: A4 portrait, 300 DPI production target.
- Interior: official logo top left, normal-text title centred, page number bottom right.
- Cover: Nursery (3+) badge allowed; interior badge prohibited.
- Visual ratio: approximately 70% activity/illustration and 30% text/support.
- Typography: rounded sans-serif, strong contrast, generous spacing.
- Activity areas: large, obvious, uncluttered, safe inside trim.
- Brand pillars: Creativity, Communication, Curiosity and Confidence only.
