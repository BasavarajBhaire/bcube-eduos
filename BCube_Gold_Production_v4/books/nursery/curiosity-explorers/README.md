# Curiosity Explorers Gold — Nursery

Book 3 of the BCube Future Skills Learning Series.

This directory is the Phase 3 source of truth for the 44-page Curiosity Explorers production package. It contains curriculum, page specifications, design and illustration standards, QA rules, publishing rules, and one self-contained production prompt per physical page.

## Release identity

- Book code: `CE`
- Level: Nursery (3+)
- Physical pages: 44
- Production standard: BCube Gold Production v5
- Canvas: A4 portrait, 2480 × 3508 px, 300 DPI
- Status: Phase 3 documentation candidate; artwork generation remains blocked until prompt QA passes.

## Non-negotiable workflow

1. Curriculum and exact page identity are locked before illustration.
2. Each page is generated from its own prompt file.
3. Existing approved brand assets are referenced, never recreated.
4. Human visual QA is mandatory before promotion to approved assets.
5. Automated checks validate structure, dimensions, hashes, numbering, and completeness; they do not make design decisions.

## Contents

- `docs/` — project, curriculum, design, illustration, QA, and publishing standards
- `production-prompts/` — P001–P044 self-contained page packages
- `qa/` — prompt-review and release-review records
- `publishing/` — release manifest and packaging instructions
