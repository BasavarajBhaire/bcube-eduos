# Final Prepress Checklist

## Asset completeness
- 44/44 pages approved
- exact physical order matches the release manifest
- page filenames and hashes recorded
- no candidate or rejected asset included

## Print properties
- A4 portrait page size
- 2480 × 3508 px source raster at 300 DPI
- correct bleed policy for selected printer
- trim-safe text, logo, activity and page number
- no clipping, corruption, transparency errors or resampling artifacts

## Content and brand
- exact titles and page numbers
- one approved BCube logo system
- approved Star mascot and illustration language
- copyright, publisher details and edition information verified

## Package
- print-ready PDF opens and renders correctly
- individual PNG package matches PDF order
- release manifest and QA report included
- printer proof reviewed before mass production
