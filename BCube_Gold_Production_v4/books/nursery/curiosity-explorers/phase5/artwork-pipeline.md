# Artwork Pipeline

1. Read the exact page prompt and Phase 3 manifest entry.
2. Resolve only locked brand assets from approved-assets.
3. Produce exactly one flat A4 portrait page.
4. Save the candidate outside approved-assets.
5. Run structural validation: filename, dimensions, DPI, transparency, page identity and integrity.
6. Complete human visual QA against the design and illustration standards.
7. Record reviewer, date, source prompt version and SHA-256.
8. Promote only the approved candidate to the release asset location.

## Prohibited

- generating multiple pages in one image
- replacing approved illustrations without explicit approval
- redrawing the BCube logo or Star mascot
- automatic promotion after technical checks alone
- overwriting an approved page without a new version and review record
