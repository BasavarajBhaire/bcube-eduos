# Phase 5 Page Review Checklist

## Identity
- exact prompt ID and title
- correct physical page and printed page number
- correct module and activity

## Brand and layout
- locked BCube logo only
- approved Star mascot model
- title, instruction banner, footer and margins match the design system
- Nursery badge appears on cover only

## Illustration
- required objects and characters are present
- no distorted anatomy, gibberish text, clipping or overlaps
- preschool-friendly scale, contrast and visual complexity
- no unapproved reuse from another book

## Activity
- instruction is short and actionable
- interaction area is large enough for Nursery children
- answer is not accidentally revealed

## Print
- 2480 × 3508 px, A4 portrait, 300 DPI metadata
- safe margins maintained
- no corruption, blur, halos or unintended transparency

Every item must be marked PASS by a human reviewer before promotion.
