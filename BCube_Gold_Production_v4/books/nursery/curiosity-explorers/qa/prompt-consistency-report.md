# Prompt Consistency Report

## Scope
Cross-page review of P001–P044 for terminology, structure, educational progression and publishing consistency.

## Required consistency rules
- Prompt IDs and physical page numbers are sequential and unique.
- Exact titles match the Phase 3 manifest and workbook.
- Module names remain consistent across prompt, curriculum and scope-and-sequence documents.
- Discovery vocabulary uses the controlled set: Observe, Compare, Explore, Classify, Predict and Question.
- Child-facing instructions remain short, direct and age appropriate.
- Teacher and parent guidance stays outside the child activity area.
- Interior pages use one logo, one title system, one instruction-banner system and one footer system.
- Cover-only elements never leak into interior pages.
- Illustration terminology stays consistent for Star, children, adults, objects and environments.
- Negative constraints are present on every prompt.

## Cross-page failure conditions
Conflicting names, duplicate concepts without progression, mismatched page numbering, inconsistent character definitions, changing brand language, repeated consecutive activity mechanics or contradictory print rules.
