# UKG V4 Production Method

The ten UKG books follow the same closed production method proven for LKG:

1. read the canonical 41-page Gold V3 source set;
2. preserve every individual source instruction, activity, expected response, teacher prompt, parent extension, illustration requirement and negative constraint;
3. migrate to the approved 44-physical-page V4 structure;
4. use UKG-specific Grade 1 readiness, reading, writing, reasoning and independence rules;
5. keep physical page 44 as the single true unnumbered back cover;
6. transparently convert contradictory source P041 Back Cover metadata into a source-backed printed-page-42 reflection;
7. validate 44 Markdown and 44 JSON packages, source lineage, numbering and deterministic locks;
8. commit, push, open a pull request, pass repository CI and merge to `main` before starting the next book.

2. Reading & Literacy Adventures
3. Maths Explorers
4. Logic & Brain Builders
