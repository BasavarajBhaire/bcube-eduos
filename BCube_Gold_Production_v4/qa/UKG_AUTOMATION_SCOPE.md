# UKG Automation QA Scope

Every UKG book build must prove:

- exactly 41 canonical Gold V3 source JSON files;
- exactly 44 physical V4 pages;
- exactly 44 Markdown and 44 JSON prompt packages;
- 40 source-backed pages with exact prompt lineage;
- printed pages 5–42 on physical pages P006–P043;
- one source-backed reflection on printed page 42;
- one true unnumbered back cover on physical page P044;
- UKG (5+) identity and Grade 1 readiness rules;
- 55–65% visual and 35–45% text/activity balance;
- deterministic official logo, approved Star and editable typography locks;
- 572 checks per completed book;
- zero critical defects;
- cumulative agenda, QA and portfolio manifests retained across all completed UKG books.
