# Communication Champions — Gold Release Candidate 1 Checklist

**Level:** Nursery  
**Release target:** Gold RC1  
**Status:** IN PRODUCTION

- [ ] Canonical source path recorded.
- [ ] Canonical source commit recorded.
- [ ] Exact page count confirmed from the repository.
- [ ] Exact page titles and visible wording confirmed from the repository.
- [ ] Publisher identity, copyright year, edition, address, email, and website approved.

## Brand assets

- [ ] `LOGO_BCUBE.PRIMARY.V1.0.0` has a real SHA-256 checksum.
- [ ] Logo dimensions and transparency are validated.
- [ ] Logo is promoted to `GOLD` in the asset registry.
- [ ] Star mascot variants required by the book are certified.
- [ ] Teacher and child character assets required by the book are certified.

## Page production

For every required page:

- [ ] One page produces one flat A4 portrait output.
- [ ] Exact versioned template resolves.
- [ ] Every asset ID resolves and matches its checksum.
- [ ] Illustration contains no final logo, page number, watermark, or invented wording.
- [ ] Final typography and layout are composed deterministically.
- [ ] Cover pages contain no interior page number.
- [ ] Visible wording exactly matches the canonical source.
- [ ] Learning purpose and child activity are clear.
- [ ] Teacher and parent guidance are included only when required by the source.
- [ ] No critical rejection condition fails.
- [ ] QA score is at least 95/100.
- [ ] Page is promoted to Gold with traceable approval metadata.

## Book assembly

- [ ] All Gold pages are assembled in the exact canonical order.
- [ ] Cover, copyright, contents, interior pages, review pages, and certificate pages are complete where required.
- [ ] Page numbering is continuous and correct.
- [ ] Typography, colour, Star, teacher, children, footer, and spacing are consistent across the book.
- [ ] No missing, duplicated, merged, cropped, or low-resolution page exists.
- [ ] Print PDF is generated from approved page artifacts only.
- [ ] Digital review PDF is generated separately where required.

## Technical preflight

- [ ] Final trim size is approved.
- [ ] Bleed and safe margins satisfy the selected printer specification.

- [ ] Founder review completed.
- [ ] Physical proof printed.
