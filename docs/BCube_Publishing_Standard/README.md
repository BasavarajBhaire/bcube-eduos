# BCube Publishing Standard (BPS)

**Status:** Founder-approved

**Current release:** BPS v1.0 – Gold Edition

The BCube Publishing Standard is the authoritative standard governing the educational, editorial, visual, technical, quality, production, and release requirements for every BCube book and related learning product.

## Scope

BPS applies to:

- Nursery, LKG, and UKG books
- Teacher resources
- Parent resources
- Worksheets, flashcards, posters, and certificates
- Print and digital publishing outputs

## Authority order

1. Canonical curriculum and book definitions in the repository
2. BCube Publishing Standard
3. Versioned templates, assets, and page manifests
4. Production prompts and implementation guidance
5. Generated outputs

AI output is never authoritative. It may create candidate illustration assets only. Final wording, branding, layout, page numbering, approval, and release remain repository-controlled.

## Governing rule

A page is not approved because it looks attractive. It is approved only when it is educationally correct, age appropriate, brand compliant, technically printable, classroom usable, and released through the defined quality gates.
